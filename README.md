# BookMarks

Simple service for save link (tag, share and etc)


### RUN

```
go run main
```


### technology stack

#### Back-End

* Go

#### Library

+ [viper](github.com/spf13/viper) - for configuration
+ [logrus](github.com/Sirupsen/logrus) - for logging
+ [chi](github.com/pressly/chi) - for routing
